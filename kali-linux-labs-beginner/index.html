root@kali:~# ./iniciar-missao_

Olá, recruta. Sou seu guia de campo. Antes de entrarmos no sistema, precisamos estabelecer as regras de engajamento. O Kali Linux é um arsenal de ferramentas digitais avançadas. Usado por operadores de elite para fortalecer defesas cibernéticas. Nossa diretriz primária é a ética. Operamos estritamente dentro de perímetros autorizados. Qualquer desvio resulta em abortar a missão. O objetivo é construir, não destruir. Entendido?

DIRETRIZ 01: AUTORIZAÇÃO

Toda operação requer autorização explícita. O uso de ferramentas em sistemas não autorizados é uma violação grave do protocolo e da lei. Mantenha-se na zona segura.

root@kali:~# ./selecionar-ambiente_

Temos duas opções para configurar seu ambiente operacional. O Ambiente Virtualizado (WSL) é o procedimento padrão para iniciantes. É rápido, seguro e isolado do seu sistema principal. A Instalação Bare-Metal é para operadores avançados que dedicam uma máquina exclusivamente para as missões.

> Modo Bare-Metal [AVANÇADO]

Implanta o Kali como sistema operacional primário.

> ALERTA: Comprometimento total do disco. Backups são mandatórios.

Ver procedimento...

1. Download: Baixe a imagem "Installer" do site oficial do Kali.

2. Pendrive Bootável: Use o Rufus ou Balena Etcher para gravar a imagem ISO em um pendrive (8GB+).

3. Configuração da BIOS/UEFI: Reinicie o computador, entre na BIOS (teclas F2, F12, Del) e configure o boot para iniciar pelo pendrive.

4. Instalação Guiada: Siga o instalador gráfico ("Graphical Install"). Na parte de particionamento, a opção mais simples é "Guided - use entire disk".

> Modo WSL [RECOMENDADO]

Executa o Kali como um subsistema dentro do Windows. Máxima eficiência, risco mínimo.

Ver procedimento...

1. Ative o WSL: Abra o PowerShell como Administrador e rode o comando:

wsl --install

Reinicie o PC se for solicitado.

2. Instale pela Microsoft Store: Abra a loja, procure por "Kali Linux" e clique em "Obter".

3. Primeira Configuração: Abra o Kali pelo Menu Iniciar. Ele pedirá para você criar um nome de usuário e uma senha. Guarde-os bem!

root@kali:~# ./comandos-basicos_

O terminal é sua interface de comando primária. Aprender a navegar é crucial. Estes são os comandos essenciais para controle e movimentação no sistema.

> sudo apt update && sudo apt upgrade -y

# Reporta a localização (diretório) atual.

> ls -la

# Lista todos os arquivos e pastas no local, com detalhes.

> cd [diretorio]

root@kali:~# ./executar-lab_

Modo de treinamento ativado. Execute as seguintes simulações em alvos designados. Lembre-se da Diretriz 01.

> FASE 1: RECONHECIMENTO

Missão 1: Scan de Portas

Use `nmap` para identificar portas abertas em `scanme.nmap.org`.

nmap scanme.nmap.org

Missão 2: Coleta de Intel

Use `whois` para obter dados públicos de `google.com`.

whois google.com

Missão 3: Enumerar DNS

Use `dig` para extrair registros DNS de `google.com`.

dig google.com

> FASE 2: ANÁLISE WEB

Missão 4: Brute-force de Diretórios

Use `gobuster` para encontrar diretórios ocultos em `testphp.vulnweb.com`.

gobuster dir -u http://testphp.vulnweb.com -w /usr/share/wordlists/dirb/common.txt

> FASE 3: ANÁLISE DE HASHES

Missão 5: Quebra de Hash MD5

Simule a quebra de um hash MD5 (`8d7...`) com `hashcat` e uma wordlist.

echo "8d788385431273d11e8b43bb78f3aa41" > hash.txt echo -e "senha\n123456\npassword" > wordlist.txt hashcat -m 0 -a 0 hash.txt wordlist.txt
