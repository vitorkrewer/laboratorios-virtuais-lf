Comando & Controle

O terminal é sua arma. Aprenda a manuseá-la.

root@kali:~# ./comandos-basicos_

O terminal é sua interface de comando primária. Aprender a navegar é crucial. Estes são os comandos essenciais para controle e movimentação no sistema.

Atualização de Sistema
sudo apt update && sudo apt upgrade -y
sudo apt update && sudo apt upgrade -y

Sincroniza os repositórios e atualiza todo o arsenal de ferramentas instaladas. Execute semanalmente.

Localização
pwd
pwd

"Print Working Directory". Mostra o caminho completo do diretório onde você está atualmente.

Listagem
ls -la
ls -la

Lista todos os arquivos (incluindo ocultos, que começam com .) e mostra detalhes como permissões e tamanho.

Navegação
cd [diretorio]
cd /home/kali

"Change Directory". Entra em uma pasta. Use cd .. para voltar um nível.
