Protocolo de Instalação

Escolha seu vetor de entrada: Virtualização ou Metal.

root@kali:~# ./selecionar-ambiente_

Temos duas opções para configurar seu ambiente operacional. O Ambiente Virtualizado (WSL) é o procedimento padrão para iniciantes. É rápido, seguro e isolado do seu sistema principal. A Instalação Bare-Metal é para operadores avançados que dedicam uma máquina exclusivamente para as missões.

RECOMENDADO

> Modo WSL

Executa o Kali como um subsistema dentro do Windows. Máxima eficiência, risco mínimo.

Procedimento:

1. Ative o WSL:

wsl --install

Reinicie o PC após o comando.

2. Instale da Store:

Abra a Microsoft Store, procure por "Kali Linux" e clique em "Obter".

3. Configuração:

Inicie pelo Menu Iniciar. Crie seu usuário e senha UNIX quando solicitado.

> Modo Bare-Metal

Implanta o Kali como sistema operacional primário. Acesso total ao hardware.

ALERTA: Requer formatação de disco. Faça backup.

Procedimento Resumido:

  • Baixe a imagem "Installer" do site oficial.
  • Grave em um pendrive (8GB+) usando Rufus.
  • Reinicie e dê boot pelo pendrive (BIOS/UEFI).
  • Siga a instalação gráfica ("Graphical Install").

Para um guia detalhado, consulte o Guia de Campo.
