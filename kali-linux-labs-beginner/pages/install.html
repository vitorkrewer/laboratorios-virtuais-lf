Instalação do Kali

Escolha o método de implantação ideal para seu ambiente.

Opção 1: WSL (Windows)

Execute o Kali Linux diretamente no Windows 10/11 sem a sobrecarga de uma máquina virtual tradicional. Ideal para iniciantes.

PowerShell (Admin)
wsl --install -d kali-linux

Opção 2: Bare Metal / ISO

Instalação completa em hardware dedicado ou VirtualBox/VMware. Acesso total ao hardware (WiFi, GPU).
