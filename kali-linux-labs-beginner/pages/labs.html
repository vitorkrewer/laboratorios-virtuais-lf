Início Instalação Terminal Laboratório Guia de Campo Videoaulas

Simulações de Campo

Execute protocolos ofensivos em ambientes controlados. Lembre-se da ética.

root@kali:~# ./executar-lab_

FASE 1: RECONHECIMENTO PASSIVO & ATIVO

Missão 1: Scan de Portas

Mapeie a superfície de ataque identificando portas abertas no alvo autorizado.

nmap scanme.nmap.org

Missão 2: Coleta de Intel

Consulte bases de dados públicas para obter informações de registro de domínio.

whois google.com

Missão 3: Enumerar DNS

Interrogue servidores DNS para descobrir a infraestrutura do alvo.

dig google.com

FASE 2: ANÁLISE DE APLICAÇÃO WEB

Missão 4: Brute-force de Diretórios

Descubra áreas ocultas e não indexadas de um site usando força bruta baseada em dicionário.

gobuster dir -u http://testphp.vulnweb.com -w /usr/share/wordlists/dirb/common.txt

FASE 3: CRIPTOGRAFIA E SENHAS

Missão 5: Quebra de Hash MD5

Recupere a senha original a partir de seu hash usando o Hashcat em modo de ataque direto.

echo "8d788385431273d11e8b43bb78f3aa41" > hash.txt echo -e "senha\n123456\npassword" > wordlist.txt hashcat -m 0 -a 0 hash.txt wordlist.txt
